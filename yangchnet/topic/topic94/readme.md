# 94.二叉树的中序遍历

## 1. 题目描述

给定一个二叉树的根节点 `root` ，返回 *它的 **中序** 遍历* 。

 

 **示例 1：** 
<img alt="" src="https://assets.leetcode.com/uploads/2020/09/15/inorder_1.jpg" style="height: 200px; width: 125px;" />
```

输入：root = [1,null,2,3]
输出：[1,3,2]

```
 **示例 2：** 

```

输入：root = []
输出：[]

```
 **示例 3：** 

```

输入：root = [1]
输出：[1]

```
 

 **提示：** 
- 树中节点数目在范围 `[0, 100]` 内
-  `-100 <= Node.val <= 100` 
 

 **进阶:** 递归算法很简单，你可以通过迭代算法完成吗？

 
**标签**
`栈` `树` `深度优先搜索` `二叉树`
