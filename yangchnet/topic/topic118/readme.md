# 118.杨辉三角

## 1. 题目描述

给定一个非负整数  *`numRows` ，* 生成「杨辉三角」的前  *`numRows`  * 行。

<small>在「杨辉三角」中，每个数是它左上方和右上方的数的和。</small>

<img alt="" src="https://pic.leetcode-cn.com/1626927345-DZmfxB-PascalTriangleAnimated2.gif" />

 

 **示例 1:** 

```

输入: numRows = 5
输出: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]

```
 **示例 2:** 

```

输入: numRows = 1
输出: [[1]]

```
 

 **提示:** 
-  `1 <= numRows <= 30` 
 
**标签**
`数组` `动态规划`
