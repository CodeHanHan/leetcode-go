# 回文数

## 1.题目描述
给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。例如，121 是回文，而 123 不是。

## 2. 示例

示例 1：
```
输入：x = 121
输出：true
```
示例 2：
```
输入：x = -121
输出：false
解释：从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
```
示例 3：
```
输入：x = 10
输出：false
解释：从右向左读, 为 01 。因此它不是一个回文数。
```
示例 4：
```
输入：x = -101
输出：false
```

## 3. 解题
1.当整数x小于0时，肯定不是回文数
2.当整数x为0时，是回文数
3.当整数x大于0时，将x反转，与x对比，相同则为回文数。
